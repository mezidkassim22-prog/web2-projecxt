Color and Background Utilities

A reference guide for applying Bootstrap's contextual colors to text and backgrounds.

  • ✓

    Text Colors: Use the .text-* classes (e.g., .text-primary) to quickly set the color of any element's text based on the theme palette.

  • ✓

    Background Colors: Use .bg-* classes (e.g., .bg-success) to apply solid background colors. Use .bg-*-subtle for lighter tints.

  • ✓

    Opacity: Text and background colors can have opacity applied using .text-opacity-* and .bg-opacity-* utilities (e.g., .text-opacity-50).

  • ✓

    Specific Colors: The primary semantic colors are: Primary, Secondary, Success, Danger, Warning, and Info.

HTML Source Code


<!-- Text Color Utilities (.text-*) -->
<p class="text-primary fw-bold">Primary Text</p>
<p class="text-success fw-bold">Success Text</p>
<p class="text-muted">Muted Text</p>

<!-- Background Color Utilities (.bg-*) -->
<div class="p-3 bg-info text-dark">
    Info Background with Dark Text
</div>

<!-- Opacity Utility (.bg-opacity-*) -->
<div class="p-3 bg-dark bg-opacity-50 text-white">
    Dark Background at 50% Opacity
</div>

<!-- Subtle Background Tints -->
<div class="p-3 bg-primary-subtle text-primary">
    Light Primary Background
</div>
