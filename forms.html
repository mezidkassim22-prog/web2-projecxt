Form Controls and Structure

Bootstrap forms provide robust styling and responsiveness for all standard form controls, including inputs, selects, and textareas.

  • ✓

    Standard Control: All text-based inputs (text, email, password, etc.), <textarea>, and <select> elements should use the .form-control or .form-select classes, respectively.

  • ✓

    Labels: Use the .form-label class on <label> elements to ensure correct styling, margins, and accessibility.

  • ✓

    Help Text: Contextual help messages should be placed under the input and use the .form-text class for appropriate styling and smaller text size.

  • ✓

    Layout: Forms are best laid out using the Grid system (rows and columns) or by using layout helpers like **Floating Labels** (see component 15).

<form>
    <!-- 1. Email Input with Help Text -->
    <div class="mb-3">
        <label for="exampleEmail" class="form-label">Email address</label>
        <input type="email" class="form-control" id="exampleEmail" 
               aria-describedby="emailHelp">
        <div id="emailHelp" class="form-text">
            We'll never share your email.
        </div>
    </div>

    <!-- 2. Textarea Control -->
    <div class="mb-3">
        <label for="exampleTextarea" class="form-label">Feedback</label>
        <textarea class="form-control" id="exampleTextarea" rows="3"></textarea>
    </div>

    <!-- 3. Checkbox Control (uses .form-check) -->
    <div class="mb-3 form-check">
        <input type="checkbox" class="form-check-input" id="exampleCheck">
        <label class="form-check-label" for="exampleCheck">Check me out</label>
    </div>
    
    <button type="submit" class="btn btn-primary">Submit</button>
</form>
