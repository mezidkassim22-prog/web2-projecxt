Dropdowns Component

Dropdowns are toggleable, contextual overlays for displaying lists of links or actions.

  • ✓

    Wrapper: Always wrap the button and the menu in a container with .dropdown (or .dropup, .dropend, etc.).

  • ✓

    Toggle: The trigger element (usually a button) must include data-bs-toggle="dropdown" and the .dropdown-toggle class.

  • ✓

    Menu Content: The menu itself uses the .dropdown-menu class, containing list items with the .dropdown-item class.

  • ✓

    Alignment: Use classes like .dropdown-menu-end to align the menu to the right of the trigger.

<!-- 1. Basic Dropdown -->
<div class="dropdown">
    <button class="btn btn-primary dropdown-toggle" type="button" 
            data-bs-toggle="dropdown" aria-expanded="false">
        Basic Dropdown
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">Action</a></li>
        <li><hr class="dropdown-divider"></li>
        <li><a class="dropdown-item" href="#">Settings</a></li>
    </ul>
</div>

<!-- 2. Split Button Dropdown -->
<div class="btn-group">
    <button type="button" class="btn btn-success">Save Changes</button>

            data-bs-toggle="dropdown" aria-expanded="false">
        <span class="visually-hidden">Toggle Dropdown</span>
    </button>
    <ul class="dropdown-menu">...</ul>
</div>
