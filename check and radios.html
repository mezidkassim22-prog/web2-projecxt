Checkboxes and Radios Component Documentation

Short Explanation of Form Checks

  • ✓

    Custom Styling: Bootstrap automatically enhances the appearance of standard browser checkboxes and radio buttons using the .form-check class wrapper.

  • ✓

    Semantics: Checkboxes (type="checkbox") allow users to select zero or more options from a list. Radio buttons (type="radio") allow users to select exactly one option from a set with the same name attribute.

  • ✓

    Layout: Use the .form-check-inline class to lay out multiple checks or radios on the same horizontal line.

  • ✓

    States: The disabled attribute works on both inputs to prevent user interaction.
