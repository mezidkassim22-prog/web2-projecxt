Layout Containers

Containers are the most basic layout element and are required when using Bootstrap's default grid system.

  • ✓

    Standard Container (.container): Sets a **responsive, fixed width** that changes at each defined breakpoint (e.g., 540px, 720px, 960px, etc.).

  • ✓

    Fluid Container (.container-fluid): Sets a width: 100%; across all breakpoints, ensuring the content always spans the **full width** of the viewport.

  • ✓

    Responsive Container (.container-{breakpoint}): A hybrid container. It starts at 100% width until the specified breakpoint is reached, where it then becomes fixed-width.

HTML Source Code


<!-- Standard Container - Fixed width after each breakpoint -->
<div class="container">
    Content contained within a responsive max-width.
</div>

<!-- Fluid Container - Always 100% width -->
<div class="container-fluid">
    Full-width content.
</div>

<!-- Responsive Container - Full width below 'lg' breakpoint, then fixed. -->
<div class="container-lg">
    Full width on small screens, then max-width on larger ones.
</div>
