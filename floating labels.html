Floating Labels

Floating labels use a container wrapper to shift a form's native <label> element into a floating placeholder.

  • ✓

    Wrapper: The core structure is a wrapper <div class="form-floating">.

  • ✓

    Order is Key: The <input> (or <textarea>/<select>) **must** come *before* the <label> element in the source code.

  • ✓

    Attributes: The input must have a valid id, and the label must have a matching for attribute. Additionally, inputs should have a placeholder.

  • ✓

    Accessibility: Using the native <label> maintains full accessibility, which is crucial for modern web development.

Live Component Preview: Floating Form Controls

HTML Source Code


<!-- Floating Input Field (Order is crucial: Input BEFORE Label) -->
<div class="form-floating mb-3">
    <input type="email" class="form-control" id="floatingEmail" placeholder="name@example.com">
    <label for="floatingEmail">Email address</label>
</div>

<!-- Floating Textarea -->
<div class="form-floating">
    <textarea class="form-control" placeholder="Leave a comment here" 
              id="floatingTextarea" style="height: 100px"></textarea>
    <label for="floatingTextarea">Comments</label>
</div>

<!-- Floating Select Menu -->
<div class="form-floating">
    <select class="form-select" id="floatingSelect">
        <option selected>Open this select menu</option>
        <option value="1">One</option>
    </select>
    <label for="floatingSelect">Works with Selects</label>
</div>
