20. Navs, Tabs, and Pills

Bootstrap's nav components provide flexible, standardized markup for building various navigation patterns like tabs, pills, and basic link lists.

  • ✓

    Base Class: All navs start with .nav on the parent <ul> or <nav> element, and each link uses the .nav-item and .nav-link classes.

  • ✓

    Tabs: Use the .nav-tabs class on the parent element for a traditional tabbed interface look.

  • ✓

    Pills: Use the .nav-pills class for rounded, button-like navigation items, often used for secondary navigation.

  • ✓

    JavaScript Integration: To make tabs/pills functional (switching panels), you need to add JavaScript attributes (data-bs-toggle="tab"/"pill") and match the link's href to the target panel's id.

HTML Source Code (Tabs with JavaScript)


<!-- 1. Tab Navigation Links -->
<ul class="nav nav-tabs" id="myTab" role="tablist">
    <li class="nav-item">
        <button class="nav-link active" id="home-tab" 
                data-bs-toggle="tab" data-bs-target="#home-tab-pane" 
                type="button">Home</button>
    </li>
    <li class="nav-item">... Profile Tab ...</li>
</ul>

<!-- 2. Tab Content Panels -->
<div class="tab-content" id="myTabContent">
    <div class="tab-pane fade show active" id="home-tab-pane" 
         role="tabpanel" aria-labelledby="home-tab">
        <!-- Content for Home -->
    </div>
    <div class="tab-pane fade" id="profile-tab-pane" >
        <!-- Content for Profile -->
    </div>
</div>

<!-- 3. Pills Example -->
<ul class="nav nav-pills justify-content-center">
    <li class="nav-item">
        <a class="nav-link active" href="#">Primary</a>
    </li>
</ul>
