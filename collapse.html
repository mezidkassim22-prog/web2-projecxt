Bootstrap Collapse Component

A detailed look at creating expanding and collapsing sections of content.

  • ✓

    Mechanism: The Collapse component relies on the Bootstrap JavaScript bundle to toggle the visibility of content.

  • ✓

    Targeting: The trigger element (usually a <button>) must use the data-bs-toggle="collapse" attribute, and its data-bs-target attribute must match the id of the content element you want to show/hide.

  • ✓

    Classes: The collapsible content block must use the base class .collapse. Add .show to make the content visible by default.

  • ✓

    Accessibility: Always include aria-expanded and aria-controls attributes on the trigger button for accessibility.

HTML Source Code


<!-- 1. The Trigger Button -->
<button class="btn btn-primary" type="button" 
        data-bs-toggle="collapse" 
        data-bs-target="#contentId" 
        aria-expanded="false" 
        aria-controls="contentId">
    Toggle Content
</button>

<!-- 2. The Collapsible Content (matches target ID) -->
<div class="collapse" id="contentId">
    <div class="card card-body">
        This content will collapse.
    </div>
</div>

<!-- Accordion Example (using data-bs-parent) -->
<div id="accordionExample">
    <div class="card">
        <button data-bs-toggle="collapse" data-bs-target="#collapseOne" >
            Item #1
        </button>
        <div id="collapseOne" class="collapse show" data-bs-parent="#accordionExample">
            ... Content for Item 1 ...
        </div>
    </div>
</div>
