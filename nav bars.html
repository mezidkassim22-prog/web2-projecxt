Navigation Bar (Navbar) Component

The Navbar is a responsive navigation header, supporting branding, links, forms, and a collapse feature for mobile screens.

  • ✓

    Core Wrapper: Use .navbar and .navbar-expand-* (e.g., `md` or `lg`) to define the base styling and the breakpoint at which the menu collapses.

  • ✓

    Branding: The .navbar-brand class is used for your logo or site name. It provides standard padding and sizing.

  • ✓

    Toggler: The responsive collapse button uses .navbar-toggler and requires data-bs-toggle="collapse" and data-bs-target (matching the collapse container's ID).

  • ✓

    Nav Links: Links are placed inside a `

      ` container, where each link uses the .nav-link class.

<nav class="navbar navbar-expand-lg bg-dark" data-bs-theme="dark">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Navbar Demo</a>
        
        <!-- Responsive Toggler: Hidden on desktop, visible on mobile -->
        <button class="navbar-toggler" type="button" 
                data-bs-toggle="collapse" data-bs-target="#navbarNavCollapse"
                aria-controls="navbarNavCollapse" aria-expanded="false" 
                aria-label="Toggle navigation">
            <span class="navbar-toggler-icon"></span>
        </button>
        
        <!-- Collapsible Content: Links and Form -->
        <div class="collapse navbar-collapse" id="navbarNavCollapse">
            <ul class="navbar-nav me-auto">
                <li class="nav-item">
                    <a class="nav-link active" href="#">Home</a>
                </li>
            </ul>

            <!-- Inline Form using .d-flex for alignment -->
            <form class="d-flex" role="search">...</form>
        </div>
    </div>
</nav>
