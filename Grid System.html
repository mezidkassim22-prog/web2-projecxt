The Grid System (Rows, Columns, Breakpoints)

The Grid System is based on a 12-column layout and uses a series of containers, rows, and columns to align and structure content.

  • ✓

    12 Columns: Each row is divided into 12 potential columns. Column classes (e.g., `col-6`) specify how many of those 12 units the column should span.

  • ✓

    Structure: Content must be placed inside a **container** (for max-width) or **container-fluid** (for full-width). Then, use `<div class="row">` inside the container, and `<div class="col-...">` inside the row.

  • ✓

    Breakpoints: Layouts are defined using five responsive tiers: **sm** (small), **md** (medium), **lg** (large), **xl** (extra large), and **xxl** (extra extra large).

  • ✓

    Inheritance: A column class applied to a larger breakpoint will apply to all smaller breakpoints unless overridden. E.g., `col-md-6` applies to screens >= 768px *and* smaller screens, unless a class like `col-sm-12` is also defined.

HTML Source Code


<!-- Always starts with a container -->
<div class="container">

    <!-- 1. Equal Width Columns Example -->
    <div class="row">
        <div class="col">1 of 3</div>
        <div class="col">2 of 3</div>
        <div class="col">3 of 3</div>
    </div>

    <!-- 2. Responsive Half-and-Half (Stacks on XS, 50% on MD and up) -->
    <div class="row">
        <div class="col-md-6">50% on Medium+</div>
        <div class="col-md-6">50% on Medium+</div>
    </div>
    
    <!-- 3. Offsetting Columns (Shift a column 4 units right on LG screens) -->
    <div class="row">
        <div class="col-lg-4 offset-lg-4">Centered Column</div>
    </div>

</div>
